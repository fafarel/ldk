Configuration LDK Admin

Assistant de configuration pour l'intégration Google Sheets

⚠️ Important : Avant de commencer, assurez-vous d'avoir un compte Google et d'être connecté à Google Cloud Console.

1Créer un Projet Google Cloud

Créez un nouveau projet dans Google Cloud Console pour gérer vos APIs.

Google Cloud Console

Actions :

  • Cliquez sur "NOUVEAU PROJET"
  • Nom : LDK-Admin-Sheets
  • Cliquez sur "CRÉER"

2Activer les APIs

Activez les APIs nécessaires pour l'intégration Google Sheets.

Google Sheets API
Google Drive API

Actions :

  • Cliquez sur chaque lien ci-dessus
  • Cliquez sur "ACTIVER" pour chaque API

3Créer les Identifiants

Créez une clé API et un Client ID OAuth 2.0.

Identifiants

3.1. Créer une Clé API

  • Cliquez sur "CRÉER DES IDENTIFIANTS" > "Clé API"
  • Copiez la clé générée

3.2. Configurer l'Écran de Consentement OAuth

  • Allez dans "APIs & Services" > "Écran de consentement OAuth"
  • Choisissez "Externe"
  • Remplissez les champs obligatoires

3.3. Créer un Client ID OAuth 2.0

  • Retournez aux "Identifiants"
  • Cliquez sur "CRÉER DES IDENTIFIANTS" > "ID client OAuth 2.0"
  • Type : "Application Web"
  • Ajoutez http://localhost aux URIs de redirection

4Configurer le Fichier

Entrez vos clés API pour générer le fichier de configuration.

Format : commence par "AIza"
Format : se termine par ".apps.googleusercontent.com"

Installez le fichier de configuration et testez l'interface admin.
